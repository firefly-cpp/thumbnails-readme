thumbnails\_readme package
==========================
This package contains the thumbnails\_readme module and is used to generate thumbnails from git repositories and add them to the README.md file.

It is tightly associated with `thumbnails-readme-action <https://github.com/KukovecRok/thumbnails-readme-action>`_.
